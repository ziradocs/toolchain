# Diagrams and Charts Examples - Flex Mode

This folder contains modern, realistic examples that demonstrate all of SlideLang's capabilities for data visualization, technical diagrams, and business analysis.

## 🎯 Main Files (Realistic Use Cases)

### `analytics_dashboard_presentation_flex.slidelang`
**Main use case** - Executive analytics dashboard:
- Complete presentation of business metrics
- Combination of charts, tables, and KPIs
- Coherent executive narrative
- Realistic use cases for decision-making

### `02_diagrams_and_charts_flex.slidelang`
**Technical reference example** covering all features:
- All chart and visualization types
- Basic diagrams and standard use cases
- Example configurations for each type

## 📚 Specialized Files (By Visualization Type)

### `02.1_data_analysis_flex.slidelang`
**Specialized in data analysis**:
- Advanced charts (bar, line, doughnut, combo, radar)
- Time-based and comparative analysis
- Business metrics and KPIs
- Business intelligence use cases

### `02.2_technical_diagrams_flex.slidelang`
**Specialized in technical diagrams**:
- Microservices architectures
- Data flows and ML pipelines
- OAuth 2.0 sequence diagrams
- State and transition diagrams
- Cloud infrastructure (AWS)
- Class models and databases

## 🚀 Types of Visualizations Covered

### Charts and Business Metrics
- **Bar Charts**: Sales, performance, comparisons
- **Line Charts**: Time-based trends, growth
- **Doughnut/Pie**: Distributions, market share
- **Combo Charts**: Multiple metrics with dual axes
- **Radar Charts**: Multidimensional evaluations

### Advanced Technical Diagrams
- **Flowcharts**: Business and technical processes
- **Sequence Diagrams**: Authentication, APIs, integrations
- **Class Diagrams**: UML data models
- **State Diagrams**: Entity lifecycles
- **Architecture Diagrams**: Distributed systems, cloud
### Maps and Geolocation
- **World Maps**: Global distribution of users/sales
- **Regional Maps**: Specific geographic analysis
- **Heatmaps**: Activity density by region
- **Markers**: Points of interest with metrics

## 🛠️ Technical Features

### Advanced Chart Configuration
- **Chart.js integration**: Full options configuration
- **Responsive design**: Adapts to different screens
- **Interactive tooltips**: Contextual information on hover
- **Custom styling**: Colors, gradients, and business styles

### Diagrams with Mermaid
- **Clear syntax**: Readable, maintainable code
- **Custom styles**: Custom CSS for different themes
- **Complex layouts**: Multi-layer architectures
- **Automatic documentation**: Generated from code
