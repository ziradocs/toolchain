# Complete Infographics

Complete infographics in ZiraDocs are comprehensive presentation templates that combine multiple slides into cohesive visual narratives. Unlike individual layouts, infographics are full presentation templates optimized for specific business scenarios with coordinated themes, validations, and export options.

## Overview

Infographics provide:
- **Complete presentation templates** for critical business scenarios
- **Coordinated visual themes** with professional color palettes and typography
- **Structured information flow** optimized for maximum impact
- **Specialized validations** ensuring content completeness
- **Multi-format export** for various use cases

## Available Infographic Types

### Business & Analytics

#### `statistical_report` - Statistical Report
Present data, research, and statistical analysis with structured metrics presentation.

## Export Formats

Infographics support multiple export formats optimized for different use cases:

```bash
# Interactive HTML (default)
slidelang build presentation.slidelang --output html

# PDF for printing and distribution
slidelang build presentation.slidelang --output pdf --infographic

# Individual images for social media
slidelang build presentation.slidelang --output images --format png

# Single page presentation
slidelang build presentation.slidelang --output single-page

# Social media optimized versions
slidelang build presentation.slidelang --output social --platforms instagram,linkedin,twitter

# Animated GIF for social sharing
slidelang build presentation.slidelang --output gif --duration 5s
```

## Content Validations

Each infographic type includes specific validations to ensure content completeness:

| Infographic Type | Required Layouts | Key Validations |
|-----------------|------------------|-----------------|
| `statistical_report` | stats, dashboard | Minimum 3 slides with data |
| `company_profile` | hero, timeline, team | Must include history and team |
| `product_launch` | hero, feature_showcase, pricing | Minimum 2 features and pricing |
| `market_analysis` | stats, comparison | Market data and competition |
| `educational_guide` | process, feature_showcase | Minimum 3 educational steps |
| `investment_pitch` | before_after, stats, pricing | Problem/solution and financials |
| `content_marketing` | hero, stats, process | Engaging content and clear CTA |
| `research_findings` | stats, comparison, dashboard | Scientific data and methodology |
| `event_promotion` | hero, feature_showcase, pricing | Event details and registration |
| `sales_funnel` | process, stats, dashboard | Conversion data and funnel stages |
| `onboarding_journey` | timeline, dashboard, process | Step-by-step guide and progress tracking |
| `crisis_communication` | before_after, process, stats | Situation assessment and action plan |
| `competitive_battlecard` | comparison, stats, process | Feature comparison and objection handling |

## Best Practices

### Visual Consistency
- Use the same theme throughout the entire infographic
- Maintain brand colors and typography consistency
- Apply consistent spacing and alignment
- Use high-quality images and graphics

### Content Strategy
- Tell a story with your data and information
- Keep text concise and scannable
- Use data visualization over raw numbers
- Include clear takeaways for your audience

### Technical Optimization
- Optimize for mobile viewing and sharing
- Include alt text for accessibility
- Use semantic markup for SEO
- Optimize load times for web sharing

### Engagement
- Include interactive elements where appropriate
- Add social sharing buttons for viral potential
- Use compelling headlines and hooks
- End with clear calls-to-action

## Dynamic Data Integration

Advanced infographics support live data integration:

```slidelang
---
mode: flex
infographic: statistical_report
theme: data_insights
data_sources: {
  api: "https://api.company.com/metrics",
  refresh_interval: "1h",
  fallback_data: "data/backup.json"
}
---

# LIVE DASHBOARD 2024
## Real-time Performance Metrics

---
layout: dashboard
---

# Current Metrics

### Sales: {{live.sales}} ({{live.sales_change}})
### Users: {{live.active_users}} ({{live.user_growth}})
### Revenue: {{live.revenue}} ({{live.revenue_change}})
```
