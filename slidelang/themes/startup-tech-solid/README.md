# Startup Tech Theme - Solid Colors

A vibrant, modern theme designed for startup tech presentations with solid color backgrounds and beautiful code block styling.

## Version
1.0.0

## Description
This is the solid color variant of the Startup Tech theme. It uses clean, solid colors instead of gradients for a more professional look while maintaining the vibrant, electric feel of the original.

## Features
- ✨ **Solid Color Design**: Clean, professional appearance with vibrant colors
- 🚀 **Electric Code Blocks**: Beautiful, glowing syntax highlighting
- 💎 **Modern Typography**: Inter font family with JetBrains Mono for code
- 🎨 **Startup-Focused**: Perfect for tech presentations, product demos, and startups
- 📱 **Fully Responsive**: Works on all screen sizes
- ⚡ **Fast Loading**: Optimized for performance

## Color Palette

### Primary Colors
- **Primary**: `#8b5cf6` (Vibrant Purple)
- **Secondary**: `#0f172a` (Deep Navy)
- **Accent**: `#06ffa5` (Electric Green)
- **Background**: `#ffffff` (Clean White)

### Functional Colors
- **Success**: `#06ffa5` (Electric Green)
- **Warning**: `#fbbf24` (Golden Yellow)
- **Danger**: `#f43f5e` (Electric Pink)
- **Info**: `#06b6d4` (Cyan Blue)

### Syntax Highlighting
- **Keywords**: `#8b5cf6` (Purple with glow)
- **Strings**: `#06ffa5` (Electric Green with glow)
- **Comments**: `#64748b` (Muted Gray)
- **Numbers**: `#fbbf24` (Golden Yellow)
- **Functions**: `#06b6d4` (Cyan Blue)

## Typography
- **Main Font**: Inter (with system fallbacks)
- **Code Font**: JetBrains Mono, Fira Code, SF Mono
- **Headings**: Inter (bold weights)

## Special Features

### Electric Code Blocks
- Glowing syntax highlighting
- Rainbow gradient header line
- Beautiful copy buttons
- Language indicators
- Line number support
- Tab group support

### Enhanced Elements
- Animated hover effects
- Gradient buttons and badges
- Glowing borders and shadows
- Modern card-style layouts
- Electric table styling

### Slide Types Support
- **Title Slides**: Dark navy background
- **Section Slides**: Purple gradient
- **Content Slides**: Clean white
- **End Slides**: Electric green
- **Closing Slides**: Purple gradient

## Usage

```bash
# Use with SlideLang CLI
slidelang build presentation.slidelang --theme startup-tech-solid

# Specify output directory
slidelang build slides.slidelang --theme startup-tech-solid --output ./dist
```

## Best Practices

### When to Use This Theme
- ✅ Startup pitch decks
- ✅ Product demonstrations  
- ✅ Technical presentations
- ✅ Developer conferences
- ✅ Code reviews and tutorials
- ✅ Modern corporate presentations

### Content Recommendations
- Use code blocks liberally - they look amazing!
- Include interactive elements and demonstrations
- Keep text concise and impactful
- Use the electric color palette for highlights
- Include progress indicators and metrics

### Design Tips
- The solid colors work great for professional settings
- Code blocks are the star - showcase your technical content
- Use the accent green for important call-to-actions
- The purple works well for branding elements
- White background ensures great readability
