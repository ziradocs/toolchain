## 🎨 Customization Examples

### 🔧 **Corporate Startup Branding**
```json
{
  "name": "my-startup-brand",
  "extends": "startup-tech",
  "variables": {
    "--primary-color": "#your-brand-purple",
    "--accent-color": "#your-brand-accent",
    "--gradient-bg": "linear-gradient(135deg, #your-color1 0%, #your-color2 100%)"
  }
}
```

### ⚡ **High Energy Variant**
```json
{
  "name": "startup-tech-neon",
  "extends": "startup-tech", 
  "variables": {
    "--accent-color": "#00ff00",
    "--syntax-string": "#00ff00",
    "--shadow-main": "0 8px 25px 0 rgba(0, 255, 0, 0.25)"
  }
}
```

### 🌙 **Dark Mode Plus**
```json
{
  "name": "startup-tech-ultra-dark",
  "extends": "startup-tech",
  "variables": {
    "--bg-content-slide": "#0f0f0f",
    "--bg-code": "#000000",
    "--text-color": "#ffffff"
  }
}
```
